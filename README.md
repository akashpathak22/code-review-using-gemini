# AI Code Reviewer
🚀 Code Review Using Gemini

An AI-powered code review tool built using MERN Stack with Google Gemini API. This project allows developers to paste their code and instantly get intelligent suggestions, improvements, and explanations generated by Gemini.

⭐ Features

🤖 AI-Powered Code Review using Google Gemini (2.5 Flash)

💡 Generates suggestions, improvements, explanations, and best practices

🧩 Supports multiple programming languages

⚡ Real-time response interface

🔐 Environment-based secure API key management

🎨 Clean and modern UI using Tailwind CSS

🔄 Optimized backend with efficient request handling

🌐 Deployed-ready full MERN architecture

🛠️ Tech Stack
Frontend

React.js

Tailwind CSS

Axios

Backend

Node.js

Express.js

Google Gemini (via @google/genai SDK)

# How to Run Locally
1. Clone Repository
git clone https://github.com/akashpathak22/code-review-using-gemini.git

2. Install Dependencies
Backend:
cd server
npm install

Frontend:
cd client
npm install

3. Start the App
Backend:
npm run dev

Frontend:
npm start
